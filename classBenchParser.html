Detailed Description

Class to parse and convert bench nodes into circuits nodes.

Bench nodes are generated by parsing ISCAS85/89/99 bench format files.

Authors
{Carolina Nogueira, Lucas Deutschmann}

std::set<label_t> BenchParser::ff_labels
private

Set containing bench node labels of all FLIP FLOP gates. When a FLIP FLOP gate is parsed, it is split into two circuit's gates: one will be handled as INPUT gate and the other one as OUTPUT gate.
